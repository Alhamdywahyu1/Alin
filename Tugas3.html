matrik invers#

Invers sendiri dapat diartikan sebagai lawan dari sesuatu (kebalikan). Jika suatu matriks memiliki invers, dapat dikatakan matriks tersebut adalah matriks nonsingular. Sebaliknya, jika suatu matriks tidak memiliki invers, maka matriks tersebut merupakan matriks singular.

Invers matriks adalah sebuah kebalikan (invers) dari kedua matriks. Apabila matriks tersebut dikalikan akan menghasilkan matriks persegi (AB = BA = |). Simbol dari invers matriks adalah pangkat -1 dan terletak di atas hurufnya. Sebagai contoh, matriks B adalah invers matriks A sehingga ditulis B = A–1 dan matriks A adalah invers dari matriks B ditulis A = B-1. Matriks A dan B merupakan dua matriks yang saling invers (berkebalikan).

sifat sifat matrik invers#

suatu matrik dapat dikatakan matriks invers dari suatu matrik adalah ketika matrik tersebut dikalikan dengan matrik invernya menghasilkan

ika A adalah matriks kuadrat, dan jika kita dapat mencari matriks B sehingga AB = BA = I, maka A dikatakan dapat dibalik (invertible) dan B dinamakan invers (inverse) dari A.

jika suatu matriks A memiliki invers (yaitu, A adalah matriks nonsingular), dan matriks B adalah inversnya, maka A dikalikan dengan B akan menghasilkan matriks identitas

mencari invers dengan eliminasi gauss jordan#

di sini saya akan mencari ivers eliminasi gauss jordan

menurut gauss jordan :

(A|I)(A-1)

Di mana A = Matriks

I = Matriks identitas

A-1 = Invers Matriks

misalnya kita mau mencari invers dari sebuah matriks dengan ordo 3x3 contoh :

\begin{bmatrix} 1 & 2 & 3 \ 0 & 1 & 4 \ 5 & 6 & 0 \end{bmatrix}

A

\begin{bmatrix} 1&2&3\ 0&1&4\ 5&6&0
\end{bmatrix}

I

\begin{bmatrix} 1&0& 0\ 0&1&0\ 0&&1 \end{bmatrix}

setelah itu kita gunakan eliminasi geuss jordan

\begin{bmatrix} 1 &2& 3|1&0&0\ 0& 1& 4|0&1&0\ 5& 6 & 0|0&0&1 \end{bmatrix}

untuk mencari invers,caranya matriks identitasnya bisa ada disebelah kiri, terserah baris berapa mau di tambahin/dikurangin/dikali/dibagi sama baris keberapa.penyelesaiannya:

sekarang kurangkan baris ketiga dengan 5x baris pertama, maksud dari baris pertama dikalikan 5 dulu baru abis itu buat ngurangin baris ke tiga hasilnya :

\begin{bmatrix} 1 &2& 3|1&0&0\ 0& 1& 4|0&1&0\ 0& -4& -15|-5&0&1 \end{bmatrix}

dari matriks diatas,tambahkan baris ke tiga dengan 4 kali baris kedua. hasilnya :

\begin{bmatrix} 1 &2& 3|1&0&0\ 0& 1& 4|0&1&0\ 0& 0& 1|5&4&1 \end{bmatrix}

selanjutnya kurangkan baris kedua dengan 4x baris ketiga. dan kita kurangkan juga baris pertama dengan 3x baris ketiga. hasilnya:

\begin{bmatrix} 1 &2& 0|16&-12&-3\ 0& 1& 0|20&-15&-4\ 0& 0& 1|-5&4&1 \end{bmatrix}

langkah terakhir tinggal kurangkan baris pertama dengan 2x baris kedua.hasilnya :

\begin{bmatrix} 1 &0& 0|-24&18&5\ 0& 1& 0|20&-15&-4\ 0& 0& 1|-5&4&1 \end{bmatrix}

yang ada disebelah kanan adalah invers dari matrik A. jadi invers matriknya:

A-1 \begin{bmatrix} -24&18&5\ 20&-15&-4\ -5&4&1 \end{bmatrix}

gauss jordan elimination juga bisa digunakan untuk mencari invers dari matriks dengan ordo yang lainnya. misalnya ordo 4x4, ordo 5x5, ordo 6x6, dan lain sebagainya.

####solusi persammaan linier dengan matrik invers

  1. Sistem Persamaan Linear: Sistem persamaan linear adalah kumpulan persamaan linear yang melibatkan beberapa variabel. Solusi dari sistem persamaan linear adalah nilai-nilai variabel yang memenuhi semua persamaan dalam sistem tersebut.

  2. Matriks Koefisien dan Vektor Hasil: Misalkan kita memiliki sistem persamaan linear dengan ( n ) persamaan dan ( n ) variabel. Matriks koefisien dari variabel-variabel ( x_1, x_2,…. x_n ) dapat kita susun menjadi matriks ( A ), dan vektor hasil dari persamaan-persamaan tersebut dapat kita susun menjadi vektor kolom ( B ).

    A = \begin{pmatrix} a_{11} & a_{12} & \ldots & a_{1n} \ a_{21} & a_{22} & \ldots & a_{2n} \ \vdots & \vdots & \ddots & \vdots \ a_{n1} & a_{n2} & \ldots & a_{nn} \ \end{pmatrix} , \

    B = \begin{pmatrix} b_1 \ b_2 \ \vdots \ b_n \ \end{pmatrix}

  3. Matriks Invers: Solusi dari sistem persamaan linear dapat ditemukan dengan menggunakan matriks invers. Jika matriks koefisien ( A ) memiliki invers, kita dapat mengalikan inversnya dengan vektor ( B ):

    [ X = A-1.B]

    Di mana ( X ) adalah vektor kolom yang berisi solusi untuk setiap variabel.

  4. Ketersediaan Invers: tidak semua matriks memiliki invers. Matriks ( A ) harus merupakan matriks persegi (jumlah baris sama dengan jumlah kolom) dan determinannya tidak boleh nol untuk memiliki invers. Jika matriks ( A ) tidak memiliki invers, maka sistem persamaan linear tersebut mungkin memiliki tidak ada solusi atau memiliki banyak solusi.

def RowSwap(A,k,l):
# =============================================================================
#     A is a NumPy array.  RowSwap will return duplicate array with rows
#     k and l swapped.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        temp = B[k][j]
        B[k][j] = B[l][j]
        B[l][j] = temp

    return B

def RowScale(A,k,scale):
# =============================================================================
#     A is a NumPy array.  RowScale will return duplicate array with the
#     entries of row k multiplied by scale.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[k][j] *= scale

    return B

def RowAdd(A,k,l,scale):
# =============================================================================
#     A is a numpy array.  RowAdd will return duplicate array with row
#     l modifed.  The new values will be the old values of row l added to
#     the values of row k, multiplied by scale.
# =============================================================================
    m = A.shape[0]  # m is number of rows in A
    n = A.shape[1]  # n is number of columns in A

    B = np.copy(A).astype('float64')

    for j in range(n):
        B[l][j] += B[k][j]*scale

    return B
